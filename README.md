# Student Management System

A console-based Student Management System developed using Java.

## Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student
* Duplicate ID Validation
* Age Validation
* Gender Validation
* Semester Validation
* Marks Validation
* Contact Number Validation
* Display Topper
* Sort Students by Marks

## Technologies Used

* Java
* OOP Concepts
* ArrayList
* NetBeans IDE
